How this works

The pipeline turns a transcript into structured metadata for archives, web applications, and demos. It builds text blocks, labels topics, generates chapters and summaries, then optionally scores engagement and extracts clips.

Batch processing uses the same configuration, so a clean first run here becomes the source of truth for the rest of the demo.
